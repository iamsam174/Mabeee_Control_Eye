# MaBeee Control Eye

## 概要
視線入力（アイトラッキング）やマウス操作などを利用して、乾電池型IoTデバイス「MaBeee（マビー）」をBluetooth経由で手軽にコントロールするWindows向けアプリケーションです。

## 特徴
- **MaBeee専用スキャン**: 起動後、PC周辺のMaBeeeだけを素早く見つけてペアリング可能です。Bluetooth特有の不要なデバイス（Unknown等）を排除したリストを表示します。
- **カメラ映像の表示**: アプリ背景にWebカメラの映像を表示しながら操作できます（視線入力アプリとの連携に最適）。
- **選べる3つの動作モード**:
  1. クリックまたは注視でタイマー実行
  2. マウスオーバー（ホバー）でタイマー実行
  3. マウスポインターがボタン内にある間だけON
- **カスタマイズ可能なUI**: ボタンのサイズ（特大〜小）、タイマー秒数（1〜180秒）、操作音の有無をご利用環境に合せて簡単に変更できます。

## 動作環境
- OS: Windows 10 / Windows 11
- Bluetooth 4.0以降対応のPC（MaBeeeとの通信に使用）
- Webカメラ（※「カメラ1」「カメラ2」などの映像を表示する場合のみ必要。不要な場合は「カメラなし」を選択可）

## 使用方法
1. MaBeeeに単4電池をセットし、コントロールしたい対象の機器（おもちゃ等）に入れます。機器の電源をオンにしてください。
2. `Mabeee_Control_Eye.exe` をダブルクリックして起動します。
3. 画面左上の「🔍 MaBeeeを探査」ボタンをクリックします。
4. 「ペアリング」のドロップダウンから検出されたMaBeeeを選択し、「接続」ボタンを押します。
5. 接続が完了したら、タイマー秒数や動作モードを設定し、「マビーON」ボタンを操作してMaBeeeを動かします。

### キーボードショートカット
- `F11` : 全画面表示のオン／オフ切り替え
- `Esc` : 全画面表示の解除
- `1` / `2` / `3` : 動作モード①・②・③の素早い切り替え
- `←` / `→` : タイマー秒数の減算／加算（1秒ずつ）

### その他操作
- **ボタンの移動位置**: 画面中央の「マビーON」ボタンは、マウスでドラッグ＆ドロップすることで画面内の使いやすい位置に自由に動かすことができます。
